Spawn redis container:
----------------------
$ sudo docker run -d -i -t --rm --name demo-redis -h demo-redis -p 3000:6379 -v /opt/demo/redis/data:/opt/redis/data -v /opt/demo/redis/log:/opt/redis/log -v /opt/demo/redis/conf/main.conf:/opt/redis/conf/redis.conf -v /opt/demo/supervisor/conf/main.conf:/etc/supervisord.conf -v /opt/demo/supervisor/conf/redis.ini:/etc/supervisord.d/redis.ini demo/redis:v1 /usr/bin/supervisord

Spawn app container:
--------------------
$ sudo docker run -d -i -t --rm --name demo-app -h demo-app -p 2000:80 --link demo-redis:demo-redis -v /opt/demo/app:/opt/app -v /opt/demo/supervisor/conf/main.conf:/etc/supervisord.conf -v /opt/demo/supervisor/conf/app.ini:/etc/supervisord.d/app.ini demo/app:v1 /usr/bin/supervisord
